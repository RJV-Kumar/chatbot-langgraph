# AI Chatbot Assistant

ChatGPT-style conversational AI application built using:

- LangChain
- LangGraph
- Streamlit
- Persistent Memory
- Session-based Chat History

## Features

- Persistent memory
- Resume previous conversations
- Context-aware responses
- Interactive UI
